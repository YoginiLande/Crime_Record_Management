# 🚨 Crime Record Management System

A web-based Crime Record Management System developed using **Python, HTML, CSS, JavaScript, and MySQL**.  
The system is designed to digitalize crime records, FIR management, and criminal data handling in an efficient and centralized way.

---

##  Project Overview

This system helps police departments maintain and manage crime-related records digitally.  
It allows easy storage, search, and tracking of FIRs, criminals, and case details.

The project uses a **Python virtual environment (venv)** for backend logic and database operations.

---

## ✨ Features

-  Admin / Police Login System  
- FIR Registration and Management  
- Criminal Record Management  
-  Case Status Tracking  
-  MySQL Database Integration    
-  Python backend with virtual environment (venv)
